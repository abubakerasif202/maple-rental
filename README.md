# Run the app locally

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and set required values (for example Stripe/JWT keys)
3. Run the app:
   `npm run dev`
